## 📊 Dataset Information

This project uses the **Pima Indians Diabetes Dataset**.

- Source: Kaggle
- Records: 768
- Features: 8 input features + 1 target
- Target:
  - 0 → No Diabetes
  - 1 → Diabetes

### Features:
- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age

### Dataset Link:
https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database
